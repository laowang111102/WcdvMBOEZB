# 医护人员排班系统 python761

## 项目概述

医护人员排班系统是基于 Spring Boot 和 Vue.js 的 Java Web 应用，旨在提供一种高效、便捷的排班解决方案。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js
- 项目管理工具：Maven
- 数据库：MySQL
- Node.js 版本：14.x
- Python 版本：2.7

## 功能模块

### 系统角色

- 管理员
- 医护人员

### 管理员功能

- 医院信息管理
  - 添加、编辑、删除医院信息
- 医护类型管理
  - 添加、编辑、删除医护类型
- 排班信息管理
  - 制定、修改、删除排班信息
- 科室管理
  - 添加、编辑、删除科室信息
- 投诉信息管理
  - 查看、处理投诉信息

### 医护人员功能

- 查看医院信息
- 查看医护信息
- 查看科室信息
- 投诉信息管理
  - 提交、查看投诉状态

## 运行环境

- Java 版本：JDK 1.8
- IDE 类型：IntelliJ IDEA 或 Eclipse
- 数据库：MySQL（5.x 或 8.x 版本）

## 目录结构

```
项目目录
├── src
│   ├── main
│   │   ├── java
│   │   └── resources
│   └── test
└── webapp
    ├── static
    └── WEB-INF
```

## 部署步骤

1. 环境准备：安装 JDK 1.8、MySQL、Node.js 14.x、Python 2.7。
2. 数据库配置：创建数据库，导入 SQL 脚本。
3. 项目构建：使用 Maven 对项目进行编译和打包。
4. 部署应用：将打包后的 war 文件部署到 Web 服务器。

## 核心亮点

- 基于成熟的 Spring Boot 和 Vue.js 技术栈，保证了系统的稳定性和性能。
- 清晰的目录结构和详尽的文档，方便开发与维护。
- 灵活的排班管理，满足不同医院科室的需求。
